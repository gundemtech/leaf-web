# leaf-web

Marketing site for Leaf — `leaf.gundem.tech`.

Astro static-site build. Tokens ported from the Leaf macOS app design system
(`Packages/LeafCore/Sources/LeafCore/UI/Tokens/` in `gundemtech/leaf`).

## Develop

```bash
pnpm install
pnpm dev    # http://localhost:4321
```

`/styleguide` shows all atomic components in both themes (toggle macOS Light/Dark to verify).

## Build

```bash
pnpm build       # → dist/
pnpm preview     # serve dist/ locally
```

## Layout

- `src/styles/` — `tokens.css` (CSS vars verbatim from app), `reset.css`, `global.css`
- `src/layouts/` — `BaseLayout` (nav + footer + OG meta) and `AuthLayout` (centered, noindex)
- `src/components/` — 13 atomic `.astro` components (`LeafButton`, `LeafCard`, `LeafNav`, …)
- `src/components/sections/` — composite landing sections (Hero, LayerA, LayerB, Surfaces, UseCases, Privacy, Pricing, FAQ, Waitlist)
- `src/pages/` — Astro file-routes
- `public/assets/icons/sprite.svg` — generated by `scripts/build-icon-sprite.sh`
- `public/assets/logo/` — PNG @1x/2x/3x (drop-in SVG replacement when ready)

## Deploy

`rsync` from `dist/` to VPS `/var/www/leaf/`. Nginx config unchanged from previous site.
