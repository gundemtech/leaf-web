- /changelog is rendered by the gundemtech/leaf-contact Worker (src/public.ts), which hand-mirrors LeafNav/LeafFooter/LeafLogo/LeafButton/LeafThemeToggle and tokens.css — if you change them here, update the mirror there.

- **Analytics: PostHog is LIVE in prod (since 2026-06-16).** Privacy-first — EU residency (eu.i.posthog.com), autocapture + session recording OFF, localStorage (no 3p cookies), DNT respected. Explicit funnel events only: `cta_clicked`, `checkout_intent` (OxaPay), `pricing_viewed`, `waitlist_submitted`, `download_clicked`, `signup_started`/`signup_completed`/`signin_completed`. Implementation: `src/components/Analytics.astro` (loaded via BaseLayout + AuthLayout) + a delegated click listener (so mirrored Nav/Button are untouched) + 4 hooks in `signup-flow.ts`. Key in Analytics.astro is the public `phc_` ingestion key (safe client-side). Full event list + UTM convention: `~/Desktop/Leaf/.agents/analytics-tracking-plan.md` (private, outside repo). Open follow-ups: enable PostHog internal-user filter (exclude our own traffic); /changelog is NOT covered (Worker-rendered — add snippet to leaf-contact if we drive traffic there); consider a reverse proxy for ad-blocked dev traffic before a big launch.
